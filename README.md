# c2
Браузерный калькулятор.

## Запуск
Откройте `index.html` или `calculator.html` в браузере.
Если нужны секретные значения для локальной разработки, создайте файл `.env` по примеру `.env.example`.

## Деплой на Vercel
1. Загрузите проект в репозиторий GitHub.
2. Импортируйте репозиторий в [Vercel](https://vercel.com/).
3. Для этого проекта не нужны команды сборки: Vercel раздаст статические файлы из корня репозитория.
4. Основная страница для деплоя: `index.html`.
